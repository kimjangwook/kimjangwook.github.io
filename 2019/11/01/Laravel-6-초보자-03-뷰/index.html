    Laravel 6 초보자 - 03. 뷰

    Laravel Laravel Tutorials
    작성일시:2019-11-01 10:38
    조회수:
    코멘트:
    1. 뷰 파일 생성
    2. 주의사항

    이전에 작성한 라우트는 Hello라는 문자열을 반환하는 단순한 라우트였다.

    웹페이지를 작성하기 위해서는 보다 복잡한 내용을 반환할 필요가 있으며, 일반적으로 우리는 일반적으로 HTML 파일을 반환한다.

    /resources/views 폴더를 확인해보자.

    폴더 안에는 welcome.blade.php라는 파일이 존재하고 있다.

    /routes/web.php에 기본적으로 작성되어 있는 라우트에서 사용한 view('welcome')는 welcome.blade.php라는 view를 불러온 것이다.

    뷰 파일 생성

    /resources/views 폴더에 hello.blade.php 파일을 생성한 이후 하단의 내용을 작성한다.

    주의사항

    라라벨은 전통적으로 폴더를 구분할때 .을 사용하였다.

    만약 hello.blade.php 파일을 resources/views/subviews라는 폴더 내에 생성하였다고 가정해보자.

    이 때의 라우트는 return view('subvies.hello')가 될 것이다.
