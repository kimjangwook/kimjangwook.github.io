    Laravel 6 초보자 - 06. 블레이드 템플릿

    Laravel Laravel Tutorials
    작성일시:2019-11-08 09:31
    조회수:
    코멘트:
    1. 개요
    2. 라우트 생성
    3. 컨트롤러 생성
    4. 컨트롤러 수정
    5. 레이아웃 생성
    6. 라우트에 해당하는 뷰 생성

    우리는 이전에 뷰파일에서 이라는 형태를 사용하여 데이터를 표현하는 방법을 간단하게 맛보았다.

    라라벨은 블레이드라는 템플릿 엔진을 사용한다.

    이번 포스트에서는 라라벨에서 블레이드를 사용하는 방법에 대해 자세히 살펴보도록 하자.

    개요

    1. 총 세개의 라우트를 만든다. 라우트는 /, /about, /services이다.

    2. 각 라우트는 각각의 컨트롤러를 가지고 있으며, 컨트롤러는 IndexController, AboutController, ServicesController 이다.

    3. 각 뷰를 표현하기 위한 레이아웃을 만든다.

    4. 지정된 라우트에 액세스 하였을 때 정해진 템플릿의 뷰를 보여준다.

    라우트 생성

    routes/web.php 파일을 다음과 같이 작성한다.

    기존에 존재하는 코드는 삭제하거나 주석처리해도 무방하다.

    컨트롤러 생성

    artisan을 통해 세개의 컨트롤러를 생성한다.

    컨트롤러 수정

    컨트롤러에는 각 라우트에 액세스 하였을 때 반환할 뷰를 지정해준다.

    1. app/Http/Controllers/IndexController.php
    1. app/Http/Controllers/AboutController.php
    1. app/Http/Controllers/ServicesController.php

    레이아웃 생성

    views/layouts/nav.blade.php

    views/app.blade.php

    라우트에 해당하는 뷰 생성

    views/index.blade.php

    views/services/index.blade.php
